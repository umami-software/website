---
title: Me
---

All informations about your session.

**Endpoints**

```text
GET /api/me
GET /api/me/teams
GET /api/me/websites
```

---

## GET /api/me

Get informations based on your auth token.

**Sample response**

```json
{
  "token": "xxxxxxxxxxxxxxx",
  "authKey": "auth:xxxxxxxxxxxxxxx",
  "shareToken": null,
  "user": {
    "id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "username": "member",
    "role": "user",
    "createdAt": "2025-10-08T18:03:19.823Z",
    "isAdmin": false
  }
}
```

## /api/me/websites

Get all your websites.

**Parameters**

- `includeTeams`: (boolean) Set to true if you want to include websites where you are the team owner.
