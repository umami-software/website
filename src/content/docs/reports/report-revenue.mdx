# Revenue

Umami Revenue allows you to view your data in a variety of ways.
Explore your data more extensively through the utilization of segments and filters.

The report works be segmenting your data into two main metrics **Visitors** and **Views**.

## Parameters

- `Website`: (required) The website you want to run the report on.
- `Date range`: (required) The date range of the report.
- `Event`: (required) The event name of the data to be aggregated.
- `Revenue Property`: (required) The property name of the number field to be aggregated.
- `User Property`: The property name of to uniquely identify the user.
