Oravo Analytics is a hosted service that requires no installation. Simply sign up for a free account to get started.

## Getting Started

### Step 1: Sign Up

Visit [analytics.imoogleai.xyz](https://analytics.imoogleai.xyz) and create your free account.

### Step 2: Add Your Website

Once logged in, add your website to start tracking analytics.

### Step 3: Install Tracking Code

Copy the provided tracking code and add it to your website's HTML.

That's it! Your analytics will start appearing in your dashboard immediately.
